# - WhSms -

Send SMS anonymously, with personalized text.

# - Installation -

`pkg install git` 

`git clone https://github.com/WhBeatZ/WhSms.git`

`pkg install bash`

`cd WhSms`

`chmod +x *`

`bash whsms.sh`

And Enjoy :)
